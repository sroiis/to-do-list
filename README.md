# Task Manager App

A simple task manager to add, manage, and keep track of your tasks.

## Features:
- Add tasks
- Display tasks
- Simple, interactive UI
- Task storage using localStorage

## How to Use:
1. Clone the repo.
2. Open `index.html` in your browser.
3. Add your tasks using the input field and button.

## Tech Stack:
- HTML
- CSS
- JavaScript
